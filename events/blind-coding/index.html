Blind Bytes

"Nothing will be shown
For the code that u own
You just need to solve
With the knowledge and skills that u evolve."


Are you up for the ultimate coding challenge? Put your skills to the test with Blind Coding! In this thrilling competition, you'll have to rely on your intuition and expertise to solve a mystery problem - all without being able to see the screen. With no internet access or communication allowed, it's a true test of your coding prowess. Do you have what it takes to become the Blind Coding champion?

INSTRUCTIONS

The event consists of two rounds:

1. Prelims

  1. This is an eliminating round.
  2. The round consists of simple coding problems.
  3. Participants will be given 20 mins to solve the given problem.

2. Mains

  1. Qualifying teams will be given problems slightly harder than the prelims round.
  2. Participants will be given 30 mins to solve the problem.

RULES

  1. Each team will have two members.
  2. Participants will be given 60 minutes to solve the problem.
  3. Participants will be given 2 problems one of easy level and another of medium level. (In total it will be 40 marks : easy (15 ) and medium (25))
  4. Participants will be allowed to turn their screen on twice for 30 seconds, but with the penalty of 3 points each time.
  5. Participants whose code gets executed successfully and has an optimal solution will be given highest preference.
  6. The second preference will be given to code with least number of errors.
  7. In case of tie time of submission will be considered.
  8. Decision made by the event organisers will be final and binding.
  9. Any arguments and/or nuisance caused to the organisers will not be tolerated.

LANGUAGES ALLOWED

  • C
  • C++
  • Java
  • Python
